# 数字电路
## 数字信号和数字电路
### 定义
```txt
                |-> 模拟电路
  电子电路分类 ---|
                |-> 数字电路

   模拟信号
     -> 时间上和幅度上都连续变化的信号

   数字信号
     -> 时间上和幅度上都断续变化的信号

   区别：
     1. 工作任务不同
        模拟电路研究的是输出与输入信号之间的大小、相位、失真等方面的关系；数字电路主要研究的是输出与输入间的逻辑关系。
     2. 三极管的工作状态不同 
        模拟电路中的三极管工作在线性放大区，是一个放大元件；数字电路中的三极管工作在饱和或截止状态，起开关作用。
```

### 模拟电路研究的问题
```txt
   模拟电路研究的问题：
     基本电路元件: 晶体三极管 ， 场效应管 ， 集成运算放大器

   基本模拟电路
     1. 信号放大及运算(信号放大、功率放大)
     2. 信号处理（采样保持、电压比较、有源滤波）
     3. 信号发生 （正弦波发生器、三角波发生器...）
```

### 数字电路研究的问题
```txt
    基本电路元件: 逻辑门电路 ， 触发器

    基本数字电路：
      1. 组合逻辑电路
      2. 时序逻辑电路(寄存器、计数器、脉冲发生器、脉冲整形电路)
      3. A/D转换器 ， D/A转换器
```

### 数字信号的描述方法
```txt
    1. 二值数字逻辑和逻辑电平
       a. 二值(0、1)数字逻辑
          表示事物状态时称二值逻辑，表示数量时称二进制数
       b. 逻辑电平： 在电路中使用 低、高 电平表示
             逻辑电平与电压值得关系(正逻辑)
             |--------|---------|----------|
             | 电压(V) | 二值逻辑 | 电平      |      
             |--------|---------|----------|  
             |   +5   |    1    | H（高电平）| 
             |   0    |    0    | L（低电平）| 
             |--------|---------|----------|
              即： 如果是+5v，表示是高电平；如果是0v，则是低电平。
    
    2. 逻辑代数
       a. 用于描述客观事物逻辑关系的数学工具，又称布尔代数或开关代数
       b. 逻辑指事物因果关系的规律
    
    3. 逻辑体制
       a. 正逻辑体制： 高电平为逻辑1 ， 低电平为逻辑0
       b. 负逻辑体制： 高电平为逻辑0 ， 低电平为逻辑1
```
